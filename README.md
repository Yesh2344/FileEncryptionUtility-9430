# File Encryption/Decryption Utility

[![Build Status](https://travis-ci.org/travis-ci/travis-web.svg?branch=master)](https://travis-ci.org/travis-ci/travis-web)
[![Code Coverage](https://coveralls.io/repos/github/coveralls/sample-repo/badge.svg?branch=master)](https://coveralls.io/github/coveralls/sample-repo?branch=master)

A React application that provides a file encryption and decryption utility.

## Installation

1. Clone the repository: `git clone https://github.com/your-username/FileEncryptionUtility.git`
2. Install dependencies: `npm install`
3. Start the application: `npm start`

## Usage

1. Select a file to encrypt or decrypt.
2. Enter a password to use for encryption or decryption.
3. Click the "Encrypt" or "Decrypt" button to perform the operation.

## API Documentation

### `encryptFile(file, password)`

Encrypts a file using the provided password.

* `file`: The file to encrypt.
* `password`: The password to use for encryption.

### `decryptFile(file, password)`

Decrypts a file using the provided password.

* `file`: The file to decrypt.
* `password`: The password to use for decryption.
